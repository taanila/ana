# ana

Moduuli **ana.py** sisältää helppokäyttöisiä funktioita kuvailevaan ja selittävään analytiikkaan. Funktioiden käytön opit **ana.ipynb**-muistion esimerkeistä.

Jos sinulla omalle koneelle asennettuna Python, Jupyter Notebook, numpy ja pandas, niin voit kokeilla esimerkkejä lataamalla tämän repositoryn omalle koneellesi: https://tilastoapu.wordpress.com/2018/11/27/github-esimerkit-omalle-koneelle/.
